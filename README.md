### **Title:**

**Flutter-Based PWA with Google Authentication**

### **Introduction:**

The goal of this project was to design a secure web application using Flutter Web that supports Google Sign-In authentication. It allows users to sign in using their Google accounts, providing both convenience and security.

### **Objective:**

* To implement Google OAuth authentication
* To build a responsive cross-platform UI
* To deploy a PWA-ready web app
* To integrate Flutter Web for seamless performance

1. **Flutter Web Setup**
2. **Firebase + Google OAuth configuration**
3. **HTML meta integration for Google Sign-In**
4. **UI implementation in Flutter**
5. **Testing on multiple devices**
6. **Deploying the web build**

### **Result:**

A working Flutter Web Application with PWA support and Google login capability.
